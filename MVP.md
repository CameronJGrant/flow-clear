# FlowClear MVP Checklist

## Must-have

- [x] **Tutorial** — 3 hand-crafted levels: basic tap, blocking (intentional fail), chain puzzle.

- [ ] **Progress persistence** — Save highest level reached. localStorage, local only.

- [x] **Sound** — Node click, fuse burn crackle, pop, win/lose chimes, UI button clicks.

- [ ] **Haptics** — Coded but needs testing on physical device (not working through RDP).

## Should-have

- [ ] **Back button handling** — Handle game state properly (e.g., dismiss win/lose overlay) instead of navigating WebView history.

## Nice-to-have (post-MVP)

- [ ] **Settings screen** — Sound toggle, reset progress.

- [ ] **Leaderboards**

- [ ] **Share / challenge friends**

- [ ] **Milestone effects** — Extra particle effects or celebrations at level milestones.

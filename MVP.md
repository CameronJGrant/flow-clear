# FlowClear MVP Checklist

## Must-have

- [x] **Tutorial** — 3 hand-crafted levels: basic tap, blocking (intentional fail), chain puzzle.

- [x] **Progress persistence** — Highest level saved to localStorage.

- [x] **Sound** — Node click, fuse burn crackle, pop, win/lose chimes, UI button clicks.

- [x] **Haptics** — Vibration feedback on taps, win/lose events. Tested on physical device.

- [x] **Settings** — Sound, vibration, dark/light mode, analytics opt-out, privacy policy link, rate us.

- [x] **Privacy policy** — Hosted at privacy.synapop.games via Cloudflare Workers.

## Should-have

- [x] **Back button handling** — Dismisses settings/win/lose overlays, exits app when nothing open.

- [x] **Release build** — Signed AAB ready for Google Play upload.

- [x] **Store assets** — Screenshots, icon, banner, descriptions in store-listing.md.

## Nice-to-have (post-MVP)

- [ ] **Leaderboards**

- [ ] **Share / challenge friends**

- [ ] **Milestone effects** — Extra particle effects or celebrations at level milestones.
